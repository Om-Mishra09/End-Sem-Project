CSS Properties Guide


What is CSS?

CSS (Cascading Style Sheets) is used to style and layout web pages by defining how HTML elements should be displayed.

It allows developers to control aspects like colors, fonts, spacing, and responsiveness, making websites more visually appealing and user-friendly.

CSS can be applied inline, internally, or externally via linked style sheets.

Inline CSS

Style attribute

Applying styles directly to an HTML element is known as inline styling.

It is useful for quick and specific changes, but is less flexible and harder to maintain.

Applying Colors

Color Property

The color property in CSS sets the text color of an element, allowing you to specify values using color names, HEX, RGB, or HSL formats.

Background-Color Property

The background-color property in CSS sets the background color of an element.

Formatting Text

Font-size Property

The font-size property in CSS sets the size of the text, using units like pixels, em, rem, or percentages.

Font-weight & Font-style Properties

The font-weight property sets text thickness, and font-style defines styles like normal or italic.

Font-family Property

The font-family property specifies the typeface for text, allowing multiple fonts as fallbacks.

line-height Property

The line-height property in CSS controls the vertical spacing between lines of text.

div element

The div element in HTML is a block-level container used to group content for styling and layout purposes, allowing developers to apply CSS styles or JavaScript functions to a set of elements collectively. We can use div to draw shapes.

Changing dimensions

Width & Height Properties

Making space

Margin Property

The margin property in CSS controls the space outside an element's border, allowing you to create distance between elements. It can be set for all sides at once or individually using margin-top, margin-right, margin-bottom, and margin-left.

Padding Property

The padding property in CSS controls the space between an element's content and its border, allowing you to create internal spacing. Like margins, it can be set for all sides or individually using padding-top, padding-right, padding-bottom, and padding-left.

Border Property

The border property in CSS sets an element's outline, defining its width, style, and color, using shorthand like border: 2px solid black;

Border-radius Property

The border-radius property in CSS creates rounded corners for an element by specifying the radius of the corners, allowing for values like pixels or percentages (e.g., border-radius: 10px;).

Margin vs Padding

margin-border-padding-content

Shorthand values for properties

Properties like margin, padding, border etc. can have multiple values corresponding to each of the sides. These values can be specified in a single line in clockwise order of top-right-bottom-left.

Margin vs Padding

Span Element

The element in HTML is an inline container used to group text or other inline elements for styling and manipulation.

Display Property

The display property in CSS defines how an element is displayed on the web page, influencing its layout and behavior. Common values include

  • Block (elements take up the full width),
  • Inline (elements take up only as much width as necessary),
  • Inline-block (combines properties of both block and inline), and
  • None (hides the element).

Compared to display: inline, the major difference is that inline-block allows to set a width and height on the element. Also, with display: inline, top and bottom margins & paddings are not respected, and with display: inline-block they are.
Now, the difference between display: inline-block and display: block is that, with display: block, a line break happens after the element, so a block element doesn't sit next to other elements

Anchor Tag

The <a> (anchor) element in HTML is used to create hyperlinks, allowing users to navigate between web pages or resources. By setting the href attribute, you can link to internal or external web pages, files, or email addresses (e.g., <a> href="https://example.com">Link</a>).

Tim Berners-Lee, the inventor of the World Wide Web, introduced the concept of hyperlinking in 1991, enabling easy navigation across interconnected web pages, forming the foundation of modern web browsing. His work revolutionized how information is shared and accessed online.

List Tags

In HTML, lists are used to group related items together, providing structure to content. There are three main types of lists:

  • Ordered List (<ol>): Displays items in a numbered sequence, useful for step-by-step instructions.
  • Unordered List (<ul>): Displays items with bullet points, ideal for non-sequential lists.
  • Definition List (<dl>): Pairs terms and descriptions, useful for glossaries or key-value structures.
List items of (<ol>) & (<ul>) are defined with (<li>) element.

What are the drawbacks of inline styling?

  1. Not reusable: You have to style each element separately, which is repetitive.
  2. Mixes content and design: It combines HTML structure with styling, making it harder to manage.
  3. Hard to update: Changing styles across the page requires manual updates for each element
  4. Tough to override: Inline styles have high priority, making them harder to change with other styles.

Embedded Styling

Style Tag & CSS Syntax

The style tag in HTML is used to define embedded CSS directly within a webpage, allowing styles to be applied to the document. Embedded styles are written inside the style tag in the head tag and affect only the current page.

Selectors

What are selectors?

Selectors in CSS are patterns used to target and apply styles to specific HTML elements.
They can be simple, like targeting an element by its tag (p for paragraphs), class (.classname), or ID (#idname), or more complex using combinations like attribute selectors or pseudo-classes (e.g., a:hover).

Element Selector

An element selector in CSS targets all instances of a specific HTML tag.

Class Selector

A class selector in CSS targets elements with a specific class attribute. It is preceded with a dot (.). Elements can have multiple classes.

ID Selector

An ID selector in CSS targets a single element with a specific id attribute. ID selector is prefixed with a hash (#). ID can not be reused and one element can have only one ID

ID for Navigation

You can use an id to navigate to a specific part of a webpage, you can create an anchor link (<a>) that points to the element with the corresponding id. When clicked, the page will scroll to the section with that id.

Comments

In HTML and CSS, comments are used to leave notes or explanations within the code, which are not displayed in the browser. They start with /* and end with */.

External CSS

Adding CSS Styles to each web page using inline or internal CSS would be too much work, if our website has hundreds of pages. Ex. Each blog in a blogging site follows the same pattern. In such a case, we use external style sheets, define styles there, and then, link those styles to each of our HTML files to put those styles on that respective web page.

How to link CSS file to HTML file?

To link a CSS file to an HTML file, you can use the tag in the HTML file's head section. The <link> tag is used to link external resources like CSS files to your HTML document.
The <link> tag has two main attributes:

  • href: Specifies the path to the CSS file.
  • rel: Specifies the relationship between the HTML file and the linked resource.
